3x-ui安装指南+搭建reality+申请ssl证书

2025-11-17 1 min read

1.先更新Debian/Ubuntu系统及安装组件
apt update -y && apt install curl sudo wget git -y

2关闭 防火墙
sudo ufw disable

3 安装3x-ui脚本
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)

4.设置管理面板端口

默认支持的HTTP端口 Cloudflare代理以下HTTP端口：80、8080、8880、2052、2082、2086、2095。

默认支持的HTTPS端口 Cloudflare代理以下HTTPS端口：443、2053、2083、2087、2096、8443。

5注册域名和绑定域名 （如果你的VPS有ipV6地址可以不用买域名直接裸跑）
将域名托管到cloudflare，没有的话 直接买。。

6.设置面板证书

输入x-ui 调出面板管理界面 选择18. SSL Certificate Management

           出现这个提示就填入域名
